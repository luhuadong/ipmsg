# ipmsg

文件描述：

```bash
├── main.c          # 主程序
├── ipmsg.h         # 协议定义
├── coms.c          # 主要的数据结构、全局变量和包的解析与生成
├── coms.h
├── encode.c        # 编码转换
├── encode.h
├── send_receive.c  # 消息、文件、文件夹的发送和接收
├── send_receive.h
├── users.c         # 用户的处理
├── users.h
├── utils.c         # 与协议无关的若干函数
└── utils.h
```
